# 任務摘要
實作碼頭前台登入驗證、會話管理與依內部廠區之資料範圍過濾。

# 來源 Spec
- 00-overview.spec.md（身份驗證與資料範圍控制）
- 03-backend.spec.md（身份驗證與資料範圍）

# 對應頁面名稱
- login

# 責任範圍
- 依後台【碼頭管理 > 碼頭人員列表】進行帳密驗證；失敗時回傳可讓前端顯示「帳號密碼輸入錯誤」之結果。
- 登入成功後建立會話，並將該人員之「內部廠區」與會話關聯，供後續查詢與寫入之資料範圍過濾。
- 支援安全登出；會話逾時時回傳未授權狀態供前端導向登入（逾時時間與續期策略：未定義）。

# 驗收標準
- 僅正確帳密可登入；登入後所有需廠區隔離之 API 僅回傳/接受該人員廠區範圍內資料。
- 登出後會話失效；逾時後請求回傳未授權。

# 備註
- 會話逾時與續期為未定義，實作時需與產品/維運協調數值並於文件中標註。
