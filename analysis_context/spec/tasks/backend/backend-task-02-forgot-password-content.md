# 任務摘要
提供「碼頭-忘記密碼」內文之讀取介面，供前台登入頁忘記密碼燈箱純顯示。

# 來源 Spec
- 00-overview.spec.md（身份驗證、後台既有功能擴增）
- 03-backend.spec.md（忘記密碼）、08) 後台既有功能擴增

# 對應頁面名稱
- login（前台燈箱）、content-block-dock-forgot-password（後台維護）

# 責任範圍
- 提供可被前台呼叫之介面，回傳「碼頭-忘記密碼」區塊內容（由後台區塊內容管理維護）。
- 與既有後台「網站管理 > 區塊內容管理」之碼頭-忘記密碼設定連動。

# 驗收標準
- 前台取得之內文與後台編輯內容一致；未設定時回傳空或預設文案依既有後台行為。

# 備註
- 無需實作重設密碼流程，僅提供顯示用內文。
