# 任務摘要
實作碼頭前台登入/登出、忘記密碼燈箱與登入後應用殼層（固定功能列、導覽）。

# 來源 Spec
- 00-overview.spec.md（身份驗證與資料範圍控制、功能模組）
- 01-routing-pages.spec.md（登入頁、進入條件）
- 02-frontend.spec.md（全域導覽與基本行為、權限/資料範圍顯示）

# 對應頁面名稱
- login、shipping（登入後預設）

# 責任範圍
- 登入頁：帳號/密碼輸入、登入送出、忘記密碼觸發與燈箱（燈箱內容由後端提供之「碼頭-忘記密碼」內文純顯示）。
- 登入失敗時顯示燈箱提示「帳號密碼輸入錯誤」。
- 登入成功後依後端回傳之資料範圍（內部廠區）僅顯示符合之資料。
- 登入後預設進入出貨作業頁面；固定功能列（登出、即時公告、出貨作業、收貨作業、退貨作業）顯示於最上方，點擊切換至對應頁面。
- 安全登出流程。

# 驗收標準
- 未登入時僅可進入登入頁，登入成功後可進入各功能頁並顯示功能列。
- 忘記密碼燈箱可開啟並顯示後端提供之內文。
- 登入失敗顯示指定錯誤提示；登出後回到登入頁。

# 備註
- 會話逾時與續期策略為未定義，前端需能處理後端回傳之未授權狀態並導向登入。
