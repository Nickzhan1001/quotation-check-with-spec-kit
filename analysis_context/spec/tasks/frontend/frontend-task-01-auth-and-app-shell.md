# Task Summary
建立前台登入流程與登入後共用外框：登入頁、忘記密碼提示（純顯示）、登出、固定頂部功能列。

# Source Spec
- analysis_context/spec/01-routing-pages.spec.md（登入頁、忘記密碼提示、共用導覽）
- analysis_context/spec/02-frontend.spec.md（登入/登出行為、共用導覽/預設頁）

# Responsibility
- Frontend：實作前台登入前/後的頁面結構與互動行為。

# Acceptance Criteria
- 存在登入頁，包含帳號/密碼輸入與送出；登入失敗可顯示燈箱訊息「帳號密碼輸入錯誤」。
- 登入頁可開啟「忘記密碼提示」燈箱/視窗，內容為純顯示並可關閉返回。
- 登入後所有頁面顯示固定頂部功能列：登出、即時公告、出貨作業、收貨作業、退貨作業。
- 登入後預設進入「出貨作業」。
- 透過頂部功能列可登出；登出後無法存取需登入頁面。

# Notes
- 會話逾時規則未定義；若後端提供逾時失效，前台需能處理被登出狀態並導回登入頁。
