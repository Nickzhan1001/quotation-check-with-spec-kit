# 任務摘要
實作即時公告列表頁與公告詳細頁，含篩選、單筆/批量已讀與刪除、狀態顯示。

# 來源 Spec
- 00-overview.spec.md（即時公告模組）
- 01-routing-pages.spec.md（公告列表頁、公告詳細頁）
- 02-frontend.spec.md（即時公告：前端互動規則）

# 對應頁面名稱
- announcement-list、announcement-detail

# 責任範圍
- 公告列表頁：篩選列、勾選、列表顯示（已讀/未讀狀態、標題、日期、操作：已讀/刪除）；批量全部已讀、全部刪除（依篩選條件）。
- 公告詳細頁：顯示日期、時間、標題、內容；刪除、返回列表。
- 列表需顯示公告已讀/未讀狀態。

# 驗收標準
- 列表支援篩選與勾選，單筆與批量已讀/刪除行為符合 Spec。
- 詳細頁正確顯示內容與操作後返回列表。
- 僅顯示符合登入者廠區之公告（依後端資料範圍）。

# 備註
- 未讀數量計算規則為未定義時，功能列上的未讀提示可依現有後端提供之欄位顯示或暫不顯示。
