# 任務摘要
實作出貨作業頁面之篩選列、碼頭儀錶板（碼頭清單與選擇車次）、未排車次列表與狀態顏色呈現。

# 來源 Spec
- 00-overview.spec.md（出貨作業管理）
- 01-routing-pages.spec.md（出貨作業頁面）
- 02-frontend.spec.md（出貨作業：篩選列、碼頭儀錶板、狀態視覺化、未排車次列表）

# 對應頁面名稱
- shipping

# 責任範圍
- 篩選列：出貨日期、班別（早班/午班/晚班）；篩選僅影響未排車次列表。
- 碼頭儀錶板：依後端回傳之啟用碼頭清單渲染；未排車次時顯示「選擇車次」下拉選單，選項格式 [車牌]([車次])(任務單號)；已選定車次時顯示車牌與通知圖示（喇叭 icon）。
- 狀態顏色：綠色 #8eba34（待出貨）、藍色 #3a9cd8（缺貨待料）、紅色 #f68480（通知未到）；純實色背景，狀態變更後 < 1 秒反映。
- 未排車次列表：格式 [車牌]([車次])(任務單號)；不顯示碼頭處理中、配送中、配送異常、配送完成；顏色規則同上。

# 驗收標準
- 篩選變更僅影響未排車次列表；碼頭儀錶板與未排車次列表資料與顏色符合 Spec。
- 所屬廠區相同之登入者看到相同碼頭儀錶板與符合條件之未排車次。

# 備註
- 車次選項與未排車次列表資料由後端依廠區與任務狀態提供。
