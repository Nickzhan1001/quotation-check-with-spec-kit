# 任務摘要
實作收貨列表頁之篩選、列表顯示、下貨狀態、下貨碼頭選擇與導航至配送點總表。

# 來源 Spec
- 00-overview.spec.md（收貨作業管理）
- 01-routing-pages.spec.md（收貨列表頁）
- 02-frontend.spec.md（收貨列表頁、排序、導航規則）

# 對應頁面名稱
- receiving-list、delivery-points-summary

# 責任範圍
- 收貨列表：顯示含收貨之配送任務，僅符合登入者廠區；下貨狀態顯示接貨中/回廠下貨/下貨完成。
- 排序：先回廠下貨再接貨中，同狀態依回報時間新到舊；下貨完成之任務車號不可點擊。
- 接貨中/回廠下貨：車號可點擊進入配送點總表。
- 回廠下貨時可選擇下貨碼頭（選項由後端回傳，已排除已選出貨/下貨碼頭）。

# 驗收標準
- 列表篩選、排序與導航符合 Spec；下貨碼頭下拉選項正確排除已選碼頭。
- 僅顯示符合廠區之任務。

# 備註
- 下貨完成在列表中的排序位置為未定義，可依回報時間或產品決策實作。
