# 任務摘要
實作退貨作業列表頁、退貨單/彙整單編輯頁、退貨單/彙整單查看頁，含篩選、彙整、顯示規則與操作入口。

# 來源 Spec
- 00-overview.spec.md（退貨作業管理）
- 01-routing-pages.spec.md（退貨作業各頁）
- 02-frontend.spec.md（退貨作業：列表、編輯、查看）

# 對應頁面名稱
- return-list、return-slip-edit、consolidated-slip-edit、return-slip-view、consolidated-slip-view

# 責任範圍
- 列表頁：搜尋/篩選（關鍵字、建立日期、客戶簡稱、溫層、退貨類型、狀態）；直退/回廠退顯示規則（依廠區）；彙整退貨單（未完成且同客戶溫層），不符時提示「請選擇同客戶溫層的未完成退貨單」；編輯/查看入口依狀態（未完成→編輯、待立單→彙整單編輯、其餘→查看）。
- 退貨單編輯/彙整單編輯：純顯示欄位與可編輯退貨資料（數量、備註、新增商品、刪除規則、檔案上傳）；儲存成功後提示並返回列表。
- 退貨單查看/彙整單查看：純顯示，返回。

# 驗收標準
- 列表篩選與顯示規則符合 Spec；彙整條件與提示正確。
- 編輯頁與後台資料連動；查看頁唯讀。

# 備註
- 商品選擇燈箱、檔案類型限制（PDF/PNG/JPG）依 PRD/後端規格實作。
