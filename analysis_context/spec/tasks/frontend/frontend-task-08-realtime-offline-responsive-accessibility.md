# 任務摘要
實作即時同步之前端訂閱/更新、網路中斷與連線恢復提示、響應式與觸控友善、無障礙（鍵盤導航、焦點、ARIA）。

# 來源 Spec
- 00-overview.spec.md（即時同步與通知、資料整合）
- 02-frontend.spec.md（即時同步/載入/斷線提示、無障礙與操作性）

# 對應頁面名稱
- 全部前台頁面（即時同步、斷線/恢復、響應式、無障礙）

# 責任範圍
- 狀態變更後自動同步（< 3 秒）；需訂閱或輪詢後端即時通道並更新 UI。
- 網路中斷時提示使用者；連線恢復時具恢復機制（具體樣式與阻擋操作範圍未定義，需合理實作）。
- 響應式與觸控：按鈕與互動元素至少 44x44px；支援平板優先與桌面/手機。
- 無障礙：鍵盤導航、焦點指示、ARIA 標籤（WCAG AA 等級）。

# 驗收標準
- 後台/多裝置操作在時限內反映至本機畫面；斷線與恢復有明確提示。
- 觸控與鍵盤可完成主要操作；焦點與標籤符合無障礙要求。

# 備註
- 離線暫存與恢復後同步之具體策略為未定義，可先實作斷線提示與重連，暫存範圍可與產品協調。
