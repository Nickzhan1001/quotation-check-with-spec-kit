# Task Summary
補齊前台跨頁即時同步的更新處理、斷線提示（含恢復後同步），並確保響應式/觸控/無障礙性約束符合規格。

# Source Spec
- analysis_context/spec/00-overview.spec.md（即時同步、可靠性非功能性摘要）
- analysis_context/spec/02-frontend.spec.md（即時同步、斷線提示、裝置斷點、44x44、無障礙性）

# Responsibility
- Frontend：即時同步更新處理、離線提示、跨裝置與可用性驗證。

# Acceptance Criteria
- 前台能在資料/狀態變更後於 < 3 秒內更新畫面（依後端同步推送或重取結果）。
- 網路中斷時能明確提示；恢復後能自動重連並更新畫面狀態。
- 在手機/平板/桌面斷點下主要頁面可使用（登入、公告、出貨、收貨、退貨）。
- 所有互動元素符合至少 44x44px 的觸控目標要求。
- 不僅依賴顏色傳遞狀態（需提供可讀替代資訊，例如文字/輔助標籤）。

# Notes
- 離線暫存（哪些操作需暫存、衝突處理）在 spec 中標示未定義；若需實作暫存佇列，需先補齊規格後才能排入開發。
